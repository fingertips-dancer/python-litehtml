# python-litehtml
## 这是做什么的？？？
这是一款基于pybind11的python package，这是对于litehtml的python实现。

python-litehtml可以解析html+css其中的元素,并计算元素所在的位置。

python-litehtml仅解析元素的位置,而不会提供具体的渲染功能。如果需要，可以搭配其他库完成渲染功能。
